Installtion Steps

  • Installtion Steps for Merchant Software
  • Steps
Step 1 -

Install Green Bill Printer Driver.

Step 2 -

Install Green Bill Merchant Desktop App/.exe.

Step 3 -

Open Green Bill Folder in Documents and open the .txt file.

Step 4 -

Enter the Printer Name which you want to link to Green Bill.

Step 5 -

Now that your printer is linked with Green Bill, you can open any document or use "Ctrl + P" .

Step 6 -

You can send the bill digitally to your customers or send bill and print at the same time.

Note -

1) The file should be less than 200 kb and it should have less than 5 pages.
2) Bill can be sent to only one customer / mobile number at one time.
